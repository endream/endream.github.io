重学JavaScript————數據類型

基本數據類型

Undefined 空值

  1. Undefined 只有一個值，就是特殊值undefined .
  2. 當使用關鍵字var 或let 聲明了一個變量但未初始化時，該變量的值就爲undefined.
  3. 引用已聲明但未初始化的變量會提示undefined ,引用未聲明的變量會直接報錯. 但是兩者在用typeof 進行判斷類型时都會返回undefined
  4. 自身未提供任何方法
    undefined 出現的情境：
  • 引用已聲明但未初始化的變量
  • 引用未定義的對象屬性
  • 執行無返回值函數
  • 執行void 表達式
  • 全局常量window.undefined 或 undefined

NULL 空指針

  1. NULL 類型同樣只有一個值，即特殊值null
  2. null 表示一個空指針對象，使用typeof 進行判斷時會返回object
  3. 使用== 比較undefined 和null 時返回true ,但嚴格意義上兩者並不相等
  4. 自身未提供任何方法

Boolean 布爾類型

  1. Boolean 擁有兩個值，true 和false ，兩者並不區分大小寫
  2. 0、空字符串、null 和undefined 都可以轉換成false

Number 數值

  1. Number 最基本的數值字面量格式是十進制整數
  2. 非嚴格模式下，以0 爲首位表示的數值是八進制整數，0x爲首位表示的數值未十六進制整數

浮點值

  • 定義浮點數必須包含小數點，且小數點後必須保留一位數字，小數點前非嚴格要求
  • 小數點後沒有數字的情況下，數值會被默認轉換成整數。小數點後只有一位且爲0時也同樣會被默認轉換成整數
  • 對於非常大或非常小的數值，會通過浮點數使用科學計數法表示。即一個數值後跟一個不限大小寫的字幕e ，再加上一個要乘的10的次冥
  • 浮點值的精確度最高可達17位小數，但在算數計算中不如整數精確。0.1 加0.2 得到的值不是0.3 , 而是0.30000000000000004

值的範圍

  • 最小的值用Number.MIN_VALUE 表示，最大的值用Number.MAX_VALUE 表示
  • 如果某個數值超出JavaScript 可以表示的範圍，則會被自動轉換成Infinity 值, 表示無窮大或無窮小

NaN Not a Number

  • 表示當前值並非數值
  • 0、+0或-0相除會返回NaN
  • 任何涉及NaN的操作都會返回NaN
  • NaN 不等於包括NaN在內的任何值
